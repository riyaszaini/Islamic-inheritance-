# **App Name**: WarithGuide

## Core Features:

- Deceased & Estate Input: Users can input details about the deceased, their marital status, and the total value of the estate to be distributed.
- Beneficiary Identification Tool: An AI-powered tool that guides users through identifying eligible heirs based on family relationships and common Islamic inheritance scenarios.
- Dynamic Share Calculation Engine: Automatically calculates each heir's individual share of the inheritance in fractions and percentages according to Fara'id (Islamic inheritance) principles.
- Interactive Distribution Summary: Displays a clear and interactive summary of the inheritance distribution, showing both fractional/percentage shares and calculated monetary values for each heir.
- Educational Insights Tool: An AI-powered tool providing brief, contextual explanations for the specific Fara'id rules applied in a given calculation, enhancing user understanding.
- Case Management: Allows users to save multiple inheritance cases and retrieve them later for review or modification, leveraging a database for persistence.
- Printable Summary Report: Generates a simple, well-formatted and printable report of the inheritance calculation and distribution summary for record-keeping.

## Style Guidelines:

- Primary color: A calm, authoritative medium blue (#2989CC), signifying guidance and trust in a serious subject like justice. It has an HSL of 200, 60%, 40%.
- Background color: A very light, desaturated blue (#ECF3F7), visually linked to the primary hue, providing a tranquil and clear canvas. It has an HSL of 200, 20%, 95%.
- Accent color: An analogous and contrasting teal-green (#4DE0BB), used for calls to action or key highlight elements to draw attention without being overwhelming. It has an HSL of 170, 70%, 60%.
- Headline and body font: 'Inter', a modern grotesque-style sans-serif, chosen for its clear, objective, and highly readable qualities across various text lengths, suiting a guidance-oriented application.
- Minimalist, clean line icons are to be used, focusing on clarity and functionality. Icons should visually represent family relationships, financial aspects, and documentation without added stylistic clutter.
- A structured and spacious layout with ample whitespace to ensure readability and reduce cognitive load. Information should be presented in digestible sections or cards, guiding the user through steps clearly.
- Subtle and functional animations should be used sparingly, primarily for transitions between input steps or when presenting calculated results, to smoothly guide the user's focus without distraction.
